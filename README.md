# ✨ Descripción

**ST BARBER** es una plataforma web orientada a la digitalización y gestión completa de barberías.  
Permite a los clientes reservar citas online y a los administradores gestionar servicios, horarios y estadísticas desde un panel intuitivo y responsive.

El proyecto está desarrollado siguiendo una arquitectura moderna basada en **Laravel 11**, utilizando buenas prácticas de desarrollo y despliegue en contenedores con Docker.

---

# 🚀 Características principales

### 👤 Clientes
- Reserva de citas online
- Modificación y cancelación de reservas
- Consulta de disponibilidad en tiempo real
- Recuperación de contraseña mediante email
- Interfaz responsive y fácil de usar

### 🛠️ Administración
- Panel de administración completo
- Gestión de clientes y citas
- Gestión de horarios y disponibilidad
- Estadísticas y métricas del negocio
- Sistema de roles:
  - Cliente
  - Administrador
